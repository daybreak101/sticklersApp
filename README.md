# sticklersApp
A full-stack restaurant ordering app built with Expo, React Native, and Firebase. Users can place an order for pickup ASAP or for a future time, browse menu, personalize their order, manage their cart, and create, login, and edit their account details.

# Tech Stack

- Firebase Authentication
- Cloud Firestore
- React Context API
- AsyncStorage
- React Hook Form
